# Project Structure

- `frontend/`: React + TypeScript + Vite + Chrome Extension(MV3) 프론트엔드
- `server/`: 추후 Nest.js 서버 구현 디렉토리(현재 placeholder)

## Frontend 실행

```bash
cd frontend
npm install
npm run dev
```

## Frontend 빌드

```bash
cd frontend
npm run build
```

Chrome 확장 로드는 `frontend/dist`를 사용하세요.
